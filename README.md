# ShellGame
Demo Shell Game inspired by Suikoden 1 mini game using Phaser 3 and Typescript

## How to run

**1.** ```npm install```

**2.** copy updated phaser typings to node_modules

**3.** ```npm run dev``` or ```npm run watch```
